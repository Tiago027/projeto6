# Meu Jogo de Texto

**Descrição:**  
Este projeto foi desenvolvido como parte do curso de Programação de Jogos Digitais no SENAI. Trata-se de um menu interativo em estilo texto, com opções de jogos simples e uma seção de créditos.

---

## 🎯 Objetivo

Criar uma aplicação simples em **Portugol Studio** que permita ao usuário interagir com diferentes jogos através de um menu.

---

## 🛠️ Ferramentas Utilizadas

- Portugol
- Git e GitHub

---

## 📜 Funcionalidades

- Jogar o **Jogo da Adivinhação**
- Jogar **Pedra, Papel e Tesoura**
- Jogar o **Quiz de Perguntas**
- Visualizar **Créditos** do projeto

---

## 🚀 Como Executar

1. Abra o **Portugol Studio**.
2. Importe o arquivo `main.alg`.
3. Execute o projeto.
4. Escolha uma opção no menu digitando o número correspondente.
